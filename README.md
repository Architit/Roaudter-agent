# Roaudter-agent (Router Agent)

Minimal skeleton for LAM multi-provider LLM Router Agent.

## Dev / Tests

- Install (dev + tests):
  - `python -m venv .venv && . .venv/bin/activate`
  - `python -m pip install -e '.[test]'`
- Run tests:
  - `PYTHONPATH=src python -m pytest -q`
